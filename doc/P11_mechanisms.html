Module P11_mechanisms

module P11_mechanisms: sig .. end
The type of kinds that applies to a given mechanism. There are two flavor of kinds: kinds that come from the standard (in particular table 34 of v2.20); and tags that we apply to groups of mechanisms.
type kind =
| Encrypt
| Sign
| SignRecover
| Wrap
| Derive
| Digest
| Generate
| Symmetric
| Asymmetric
| DES
| DES3
| AES
| RSA
| DH
| EC 
val kinds : Pkcs11.CK_MECHANISM_TYPE.u -> kind list
Return the list of kinds that are defined for a mechanism. The list may be empty.
val is : kind list -> Pkcs11.CK_MECHANISM_TYPE.u -> bool
Checks that a mechanism as *all* the kinds present in the list.
exception Mechanism_not_supported of string
Support missing for a given mechanism in key_type.
val key_type : Pkcs11.CK_MECHANISM_TYPE.u -> Pkcs11.CK_KEY_TYPE.u option
Return the key type associated to a given mechanism. May raise Mechanism_not_supported because the implementation of key_type is partial. Some mechanisms do not have associated key types (e.g., hash algorithms).
